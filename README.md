Vortex Quake 2
=====

Vortex Quake 2 is an RPG mod for Quake 2.
This repository contains an updated version of the source code, sanctioned by GHz. It is a merger of a fork of 4.6 code, vrxcl code, and originally unreleased code of the original 4.7/4.8 versions of Vortex.
